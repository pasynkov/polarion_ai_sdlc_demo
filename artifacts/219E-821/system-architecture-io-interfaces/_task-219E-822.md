## System Architecture & I/O Interfaces

Covers the machine/sub-system architecture including boundary definitions, electrical connections, CAN network topology, wiring harness, input components (HMI, joysticks, sensors, switches, pedals), and output components (hydraulic hardware, electronic components, displays). Defines signal ranges, pin assignments, and inter-component interfaces.
