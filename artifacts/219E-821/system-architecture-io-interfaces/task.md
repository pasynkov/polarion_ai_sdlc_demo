## Create System Architecture & I/O Interface Specification Document

Produce a complete system architecture document that includes: a boundary diagram of the machine/sub-system, a wiring harness overview, an electrical connection table with pin assignments, CAN network topology diagram with message list (J1939 standard and proprietary), and tables for all input components (HMI, sensors, switches, pedals) and output components (hydraulic hardware, electronic components, displays) with their respective signal ranges and operating conditions.
