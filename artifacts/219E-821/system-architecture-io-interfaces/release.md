## Release: System Architecture & I/O Interface Specification v1.0

Initial release of the system architecture and I/O interface specification covering the machine/sub-system boundary definition, complete electrical connection tables, CAN network configuration, and signal range specifications for all input and output components. This release is a prerequisite for software integration and hardware-in-the-loop (HIL) test setup.
