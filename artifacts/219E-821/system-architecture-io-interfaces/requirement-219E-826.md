## System Architecture & I/O Interface Boundary Requirements

The system shall define clear physical and logical boundaries for the machine/sub-system, including a complete wiring harness description, all electrical connections with pin assignments, CAN network topology (including J1939 standard messages and any proprietary message definitions), and a full enumeration of input components (HMI joysticks, displays, sensors, switches, pedals with expected signal ranges such as voltage and frequency) and output components (hydraulic hardware including pumps, motors, and valves; electronic components including lights and buzzers; and displays with required output current ranges).
