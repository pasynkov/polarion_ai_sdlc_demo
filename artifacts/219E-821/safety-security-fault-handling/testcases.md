## Verify Fault Detection, Safety State Transitions, and Error Reporting

Verify that the software correctly detects all defined fault conditions within the specified FTTi, transitions to the appropriate error level (Warning, Limited, Safe state) for each fault, generates the correct SPN/FMI error codes, displays the correct blink codes on the HMI, and that safety-critical functions remain within the targeted SIL/PL requirements under both systematic and random hardware fault injection scenarios.
