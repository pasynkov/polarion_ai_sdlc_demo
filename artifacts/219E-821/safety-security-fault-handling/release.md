## Release: Safety, Security & Fault Handling Requirements Specification v1.0

Initial release of the safety and fault handling requirements including HARA/FMEA-derived safety requirements, SIL/PL targets, FTTi values, fault reaction matrix (SPN/FMI codes, error levels, blink codes), and cybersecurity level targets. This release is a prerequisite for functional safety planning (FSP) and software safety architecture design.
