## Safety, Security & Fault Handling

Covers functional safety requirements derived from hazard and risk analysis (HARA) and FMEA results, targeted SIL/PL levels, fault tolerance time intervals, and functional safety overviews. Also encompasses cybersecurity level targets, as well as fault detection, error reaction, error reporting (SPN/FMI), error severity levels (Warning, Limited, Safe), and blink codes.
